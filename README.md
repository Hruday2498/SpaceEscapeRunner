# Space Escape Runner

>Space Escape Runner is a simple 2D endless runner game built using React Native and Expo. The objective is to
avoid incoming asteroids and survive as long as possible while increasing your score.

## Features

- Endless gameplay
- Score tracking
- Random asteroid generation
- Collision detection
- Simple and responsive user interface

## Built With

- React Native
- Expo
- JavaScript

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Hruday2498/SpaceEscapeRunner.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npx expo start
```

## Build APK

```bash
eas build --platform android --profile preview
```
